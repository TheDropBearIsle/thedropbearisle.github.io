Troubleshooting
I can’t see the server in the browser
Try refreshing, double-check spelling, and confirm you’re on the correct Evrima version. If it still won’t show, check Discord announcements for maintenance or downtime.
High ping / lag
AU/OCE players should see the best ping. If you’re outside the region, expect higher latency. Discord announcements will also note if we’re experiencing server strain during peak events.
How do I report a rule break?
Use the Discord reporting process and include evidence (clips/screenshots, timestamps, player names). The more detail, the faster we can help.
